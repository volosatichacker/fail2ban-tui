# 🛡️ Интерактивная TUI Панель для Fail2ban

[![GitHub license](https://img.shields.io/github/license/volosatichacker/fail2ban-tui?style=flat-glowing&color=green)](LICENSE)
[![GitHub stars](https://img.shields.io/github/stars/volosatichacker/fail2ban-tui?style=flat-glowing)](https://github.com/volosatichacker/fail2ban-tui/stargazers)
[![Shell Script](https://img.shields.io/badge/shell-bash-4ea1a3?style=flat-glowing&logo=gnu-bash)](https://www.gnu.org/software/bash/)

Легковесная, высокопроизводительная и визуально приятная интерактивная консольная панель (TUI) для мониторинга и управления службой **[Fail2ban](https://github.com/fail2ban/fail2ban)** (ссылка на официальный репозиторий) непосредственно из командной строки.

---

## 🚀 Возможности

- 📊 **Мониторинг службы в реальном времени**: Мгновенно показывает статус `fail2ban-server` (активен/остановлен).
- 📂 **Динамическое определение Jails**: Автоматически сканирует и выводит список всех запущенных правил (например, `sshd`, `nginx-http-auth`).
- 🌍 **Интеграция с Geo-IP**: Выполняет онлайн Geo-IP запросы для забаненных IP, отображая их **Страну** и **Провайдера/Организацию** через сервис `ip-api.com`.
- 🔨 **Управление банами на лету**:
  - **Разбан (Unban)**: Легко снимайте блокировку с IP-адресов прямо из списка.
  - **Бан (Ban)**: Вручную добавляйте нежелательные IP в бан-лист выбранного jail.
- 📡 **Сбор забаненных IP**: Автоматическая отправка каждого забаненного IP (как вручную через TUI, так и автоматически службой в фоне) на централизованный эндпоинт (`https://banned.norkn.ru/api/ban`) с помощью встроенного действия Fail2ban.
- 📜 **Живые логи**: Потоковый просмотр последних записей из файла `/var/log/fail2ban.log`.
- 🗣️ **Мультиязычность**: Полная поддержка **Русского** и **Английского** языков с возможностью переключения прямо в меню панели.
- 🎨 **Красивое оформление**: Упорядоченные цветные таблицы, согласованные палитры цветов ANSI для удобного чтения.

## 🔥 Последние обновления (v1.3.0)

*   📡 **Интеграция со сбором забаненных IP**: Автоматически создает кастомное действие Fail2ban (`norkn-ban`) и регистрирует его в `jail.local` для автоматической отправки заблокированных IP на `banned.norkn.ru`.
*   🌐 **Мультиязычный TUI**: Интерактивный выбор языка (Русский/Английский) при первом запуске с возможностью переключения языка прямо из меню панели.
*   ⚙️ **Управление настройками Jails**: Динамическое изменение параметров `bantime`, `findtime` и `maxretry` в памяти с сохранением в конфигурационный файл `/etc/fail2ban/jail.local` при помощи автоматического парсера/обновителя INI.
*   📦 **Автоматическая установка зависимостей**: Если Fail2ban отсутствует в системе, скрипт предложит пользователю установить его автоматически и настроит безопасный базовый профиль конфигурации.
*   🎨 **Исправление визуального выравнивания**: Исправлена обработка управляющих ANSI-последовательностей в таблицах для достижения идеального выравнивания колонок.

## 🛠️ Установка и использование

Для установки панели на любой сервер выполните одну простую команду:

```bash
sudo curl -fsSL -o /usr/local/bin/fail2ban https://raw.githubusercontent.com/volosatichacker/fail2ban-tui/main/fail2ban && sudo chmod +x /usr/local/bin/fail2ban && sudo fail2ban
```

После установки вы можете вызывать панель из любого места одной командой:

```bash
sudo fail2ban
```

## ⚙️ Требования

> [!IMPORTANT]
> Скрипт должен запускаться с привилегиями **root** (через `sudo`), чтобы иметь права на обращение к `fail2ban-client` и чтение системных логов.

### Зависимости
- `curl` — используется для отправки запросов к API Geo-IP.
- `jq` (Необязательно, но рекомендуется) — используется для быстрого парсинга JSON (если отсутствует, скрипт автоматически переключится на встроенный регулярный поиск).
