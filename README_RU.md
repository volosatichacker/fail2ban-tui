# 🛡️ Интерактивная TUI Панель для Fail2ban

[![Shell Script](https://img.shields.io/badge/shell-bash-4ea1a3?style=flat-glowing&logo=gnu-bash)](https://www.gnu.org/software/bash/)

Легковесная, высокопроизводительная и визуально приятная интерактивная консольная панель (TUI) для мониторинга и управления службой **Fail2ban** непосредственно из командной строки.

---

## 🚀 Возможности

- 📊 **Мониторинг службы в реальном времени**: Мгновенно показывает статус `fail2ban-server` (активен/остановлен).
- 📂 **Динамическое определение Jails**: Автоматически сканирует и выводит список всех запущенных правил (например, `sshd`, `nginx-http-auth`).
- 🌍 **Интеграция с Geo-IP**: Выполняет онлайн Geo-IP запросы для забаненных IP, отображая их **Страну** и **Провайдера/Организацию** через сервис `ip-api.com`.
- 🔨 **Управление банами на лету**:
  - **Разбан (Unban)**: Легко снимайте блокировку с IP-адресов прямо из списка.
  - **Бан (Ban)**: Вручную добавляйте нежелательные IP в бан-лист выбранного jail.
- 📜 **Живые логи**: Потоковый просмотр последних записей из файла `/var/log/fail2ban.log`.
- 🗣️ **Мультиязычность**: Полная поддержка **Русского** и **Английского** языков с возможностью переключения прямо в меню панели.
- 🎨 **Красивое оформление**: Упорядоченные цветные таблицы, согласованные палитры цветов ANSI для удобного чтения.

## 🛠️ Установка и использование

Для установки панели на любой сервер выполните одну простую команду:

```bash
sudo curl -fsSL -o /usr/local/bin/fail2ban https://raw.githubusercontent.com/volosatichacker/fail2ban-tui/main/fail2ban && sudo chmod +x /usr/local/bin/fail2ban && sudo fail2ban
```

После установки вы можете вызывать панель из любого места одной командой:

```bash
sudo fail2ban
```

## ⚙️ Требования

> [!IMPORTANT]
> Скрипт должен запускаться с привилегиями **root** (через `sudo`), чтобы иметь права на обращение к `fail2ban-client` и чтение системных логов.

### Зависимости
- `curl` — используется для отправки запросов к API Geo-IP.
- `jq` (Необязательно, но рекомендуется) — используется для быстрого парсинга JSON (если отсутствует, скрипт автоматически переключится на встроенный регулярный поиск).
